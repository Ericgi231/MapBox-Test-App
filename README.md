# MapBox Test App

A sandbox application for testing and exploring MapBox APIs.

## Overview

This project serves as a testing environment for MapBox API functionality, featuring:
- **Frontend**: React
- **Backend**: Python FastAPI

## Purpose

This is a development sandbox for experimenting with various MapBox features and API endpoints.

## Tech Stack

- React (Frontend)
- FastAPI (Backend)
- MapBox APIs

## Getting Started

### Prerequisites

- Node.js and npm
- Python 3.7+
- MapBox API key

### Installation

1. Clone the repository
2. Install frontend dependencies:
  ```bash
  cd frontend
  npm install
  ```
3. Install backend dependencies:
  ```bash
  cd backend
  pip install -r requirements.txt
  ```

### Running the Application

**Backend:**
```bash
cd backend
uvicorn main:app --reload
```

**Frontend:**
```bash
cd frontend
npm start
```
